\subsection{Description détaillée}
\hyperlink{class_tag}{Tag} file parser.

Reads an X\+M\+L-\/structured tagfile and builds up the structure in memory. The method \hyperlink{class_tag_file_parser_a92748cf53ec12acd05e1003b1fcd34e1}{build\+Lists()} is used to transfer/translate the structures to the doxygen engine.
