\subsection{Description détaillée}
Represents a stack of variable to class mappings as found in the code. Each scope is enclosed in \hyperlink{class_variable_context_ae1d858d3ad4778e79ac69a021474edab}{push\+Scope()} and \hyperlink{class_variable_context_a648ebfb9146a61b70dffe684aec77940}{pop\+Scope()} calls. Variables are added by calling add\+Variables() and one can search for variable using \hyperlink{class_variable_context_ac6643c9a7fbfa30d341efe5c62c862fb}{find\+Variable()}.
