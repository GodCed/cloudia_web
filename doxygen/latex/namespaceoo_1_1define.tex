Extension to Tcl\+O\+O to add static methods. Defines the method on the class instead of on the object. Can be used for the creation of megawidgets using Tcl\+O\+O by overriding the unknown method to detect if the user is trying to instantiate a widget (because the method will be unknown and start with a dot).
