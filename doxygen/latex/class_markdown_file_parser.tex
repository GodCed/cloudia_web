Starts processing a translation unit (source files + headers). After this call \hyperlink{class_markdown_file_parser_a56e4994924ce8ac7710e2a1da0f98807}{parse\+Input()} is called with same\+Translation\+Unit set to F\+A\+L\+S\+E. If \hyperlink{class_markdown_file_parser_a56e4994924ce8ac7710e2a1da0f98807}{parse\+Input()} returns additional include files, these are also processed using \hyperlink{class_markdown_file_parser_a56e4994924ce8ac7710e2a1da0f98807}{parse\+Input()} with same\+Translation\+Unit set to T\+R\+U\+E. After that \hyperlink{class_markdown_file_parser_adaf4aeedc27abe4dd8cc4a58d59b298f}{finish\+Translation\+Unit()} is called.
