\subsection{Description détaillée}
The \hyperlink{class_q_g_list}{Q\+G\+List} class is an internal class for implementing Qt collection classes. 

\hyperlink{class_q_g_list}{Q\+G\+List} is a strictly internal class that acts as a base class for several \hyperlink{}{collection classes}; \hyperlink{class_q_list}{Q\+List}, \hyperlink{class_q_queue}{Q\+Queue} and \hyperlink{class_q_stack}{Q\+Stack}.

\hyperlink{class_q_g_list}{Q\+G\+List} has some virtual functions that can be reimplemented to customize the subclasses. 
\begin{DoxyItemize}
\item \hyperlink{class_q_g_list_a9a9c5fa3888371979c12e3207b3f9ec5}{compare\+Items()} compares two collection/list items. 
\item \hyperlink{class_q_g_list_a92e5a02569162c5951a90bca606223a2}{read()} reads a collection/list item from a \hyperlink{class_q_data_stream}{Q\+Data\+Stream}. 
\item \hyperlink{class_q_g_list_a03713abae9948cba74cfadfd4b0bd4f7}{write()} writes a collection/list item to a \hyperlink{class_q_data_stream}{Q\+Data\+Stream}. 
\end{DoxyItemize}Normally, you do not have to reimplement any of these functions. If you still want to reimplement them, see the \hyperlink{class_q_str_list}{Q\+Str\+List} class (qstrlist.\+h), which is a good example.
