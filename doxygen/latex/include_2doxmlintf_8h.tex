\subsection{Description détaillée}
The interface to the object model provided by the X\+M\+L parser library. 

To start using this library one calls \hyperlink{include_2doxmlintf_8h_addc96d6d7f1823dbe573dd703865568c}{create\+Object\+Model()} and then uses the returned \hyperlink{class_i_doxygen}{I\+Doxygen} interface to read doxygen generated X\+M\+L output and navigate through the information contained in it.
