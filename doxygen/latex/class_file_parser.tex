Starts processing a translation unit (source files + headers). After this call \hyperlink{class_file_parser_a129994a05caded4f7bfe5d30945f008b}{parse\+Input()} is called with same\+Translation\+Unit set to F\+A\+L\+S\+E. If \hyperlink{class_file_parser_a129994a05caded4f7bfe5d30945f008b}{parse\+Input()} returns additional include files, these are also processed using \hyperlink{class_file_parser_a129994a05caded4f7bfe5d30945f008b}{parse\+Input()} with same\+Translation\+Unit set to T\+R\+U\+E. After that \hyperlink{class_file_parser_a08ab809441600d2ace9508c040bc3943}{finish\+Translation\+Unit()} is called.
