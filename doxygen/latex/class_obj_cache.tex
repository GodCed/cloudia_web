\subsection{Description détaillée}
Cache for objects. 

This cache is used to decide which objects should remain in memory. It uses a least recently used policy (L\+R\+U) to decide which object should make room for a new object when the cache is full. An object should be added using \hyperlink{class_obj_cache_affd025326493c40ac4e7f5b4c0c3ad9b}{add()}, and then \hyperlink{class_obj_cache_ac4d9eb810e8f6ca2fbd4cf058ff3fe0e}{use()} should be called when the object is used.
