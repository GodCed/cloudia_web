\subsection{Description détaillée}
This scanner is used to convert a string into a list of function or template arguments. Each parsed argument results in a \hyperlink{struct_argument}{Argument} struct, that is put into an \hyperlink{class_argument_list}{Argument\+List} in declaration order. Comment blocks for arguments can also be included in the string. The argument string does not contain new-\/lines (except inside any comment blocks). An \hyperlink{struct_argument}{Argument} consists of the string fields\+: type,name,default value, and documentation The \hyperlink{struct_argument}{Argument} list as a whole can be pure, constant or volatile.

Examples of input strings are\+: 
\begin{DoxyCode}
\textcolor{stringliteral}{"(int a,int b) const"}
\textcolor{stringliteral}{"(const char *s="}hello world\textcolor{stringliteral}{",int=5) = 0"}
\textcolor{stringliteral}{"<class T,class N>"}
\textcolor{stringliteral}{"(char c,const char)"}
\end{DoxyCode}


Note\+: It is not always possible to distinguish between the name and type of an argument. In case of doubt the name is added to the type, and the match\+Argument\+List in \hyperlink{util_8cpp}{util.\+cpp} is be used to further determine the correct separation.
