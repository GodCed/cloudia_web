\subsection{Description détaillée}
Singleton for configuration variables.

This object holds the global static variables read from a user-\/supplied configuration file. The static member \hyperlink{class_config_ad62b66142ca8c837019cd04de34c8caa}{instance()} can be used to get a pointer to the one and only instance.

Set all variables to their default values by calling \hyperlink{class_config_ad62b66142ca8c837019cd04de34c8caa}{Config\+::instance()}-\/$>$\hyperlink{class_config_a91bcb187ed95ec673137e413a4d77203}{init()}
