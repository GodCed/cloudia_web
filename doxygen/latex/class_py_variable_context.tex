\subsection{Description détaillée}
Represents a stack of variable to class mappings as found in the code. Each scope is enclosed in \hyperlink{class_py_variable_context_a0546bb3cb235db78d2e8fcbb1dcb6502}{push\+Scope()} and \hyperlink{class_py_variable_context_ae24b90d1862e5ea27664f1d5adbf215a}{pop\+Scope()} calls. Variables are added by calling add\+Variables() and one can search for variable using \hyperlink{class_py_variable_context_a9fc4eb25a82e2566d2b6bdab62fbd5d7}{find\+Variable()}.
