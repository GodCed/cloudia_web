\subsection{Description détaillée}
The \hyperlink{class_store}{Store} is a file based memory manager. 

You can open the store using \hyperlink{class_store_a9707050019d12be2db8ea1b38d535f03}{open()}. Then obtain a handle via \hyperlink{class_store_ac1ce3fb86b3bc46125756b9ef782c095}{alloc()} followed by a sequence of \hyperlink{class_store_aec754c74cf190cb55281a923db21194a}{write()} commands to store information, and finalize it using \hyperlink{class_store_a616a1f53fa04d39d505a03b1f49ab7a6}{end()}.

Later on you locate the information with \hyperlink{class_store_ad77053023f371abfc70f76ee2a14c377}{seek()} using the handle obtained with \hyperlink{class_store_ac1ce3fb86b3bc46125756b9ef782c095}{alloc()}, and then use a sequence of \hyperlink{class_store_a02df7a98367838e436f8f8b3367016d1}{read()} calls to read the information back.

If no longer needed the storage space can be freed using \hyperlink{class_store_aed5b8a83aa15dfc3f7f1ca930a4d2583}{release()}.

The store will dynamically grow the file on disk if needed.
